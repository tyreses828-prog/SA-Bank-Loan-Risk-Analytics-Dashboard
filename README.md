# SA Bank Loan Risk Analytics Dashboard

## Overview
End-to-end financial data analysis project 
simulating SA bank loan risk assessment.

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Key Findings
- Random Forest: 72% accuracy, AUC 0.720
- Top default predictor: Credit Score (19.1%)
- High risk threshold: Credit score below 580
